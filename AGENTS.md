## Agent skills

### Issue tracker

Issues and PRDs live as local Markdown under `.scratch/<feature-slug>/`. No external PR triage (solo project). See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`) recorded as `Status:` lines in issue files. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout: root `CONTEXT.md` plus `docs/adr/`. GStreamer platform sinks and VideoOverlay integration are documented there. See `docs/agents/domain.md`.
